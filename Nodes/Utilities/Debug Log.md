# Debug Log Node

![Debug-Log-Node](/IMG/Debug-Log-Node.png)

## About

This node writes the given text to the Debug Console

## Inputs

| Input   | Type   | Description                           |
| ------- | ------ | ------------------------------------- |
| Print   | Event  | Triggers a write to the Debug Console |
| Message | String | The message to write                  |
