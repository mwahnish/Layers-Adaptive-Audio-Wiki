# Magnitude

![Magnitude-Node](/IMG/Magnitude-Node.png)

## About

This node takes a Vector3 as input, and outputs the length of the vector. The length of a vector is the square root of (x\*x + y\*y+z\*z)

## Inputs

| Input | Type    | Description          |
| ----- | ------- | -------------------- |
| Input | Vector3 | The incoming Vector3 |

## Outputs

| Output | Type  | Description              |
| ------ | ----- | ------------------------ |
| Output | Float | The length of the vector |
